# WebGIS SPPG Kota Semarang

## Deskripsi
WebGIS ini dibuat untuk menganalisis keterjangkauan SPPG terhadap fasilitas pendidikan (SD, SMP, SMA) di Kota Semarang.

## Fitur
- Visualisasi SD, SMP, SMA, dan SPPG
- Isochrone (analisis keterjangkauan)
- Heatmap kepadatan sekolah
- Analisis coverage
- Dashboard chart (jumlah sekolah & siswa)
- Rekomendasi lokasi SPPG

## Teknologi
- HTML, CSS, JavaScript
- MapLibre GL JS
- Turf.js
- Chart.js

## Data
- Data sekolah (SD, SMP, SMA)
- Data SPPG
- Batas kota Semarang

## Cara Menjalankan
Buka file `index.html` menggunakan Live Server

## Output
- WebGIS interaktif
- Dashboard analisis
- Rekomendasi lokasi SPPG
